DOM and BOM

Browser Object Modle represents the window or tab. Document Object Model represents the web page. BOM accessable through window object. window has document property. window object also has othe useful properties.

The browser converts HTML docs into a DOM whos structure is represented by a NODE tree. Nodes can be elements, attributess of elements or anythong that is part of the HTML doc. document.something serves as the entry point to the node tree

DOM selectors are ues to access HTML elements in the document. Once we have accessed an element we can change things about it dynamically throught js by accessing its properties

we can also create new elements using docment.createElement('element') The created elements must be .append ed to their parent element before they will show up on the page
